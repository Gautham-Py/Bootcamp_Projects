# 🔥 F.L.A.M.E.S

A small browser-based version of the classic **FLAMES** name game.

## How to Use

1. Open **index.html** in any browser.
2. Enter two names.
3. Click **FLAMES**.
4. The result will appear after a short animation.

The game will automatically reset after a short time, or you can press **Clear** to start again.

## Notes

* Works completely offline
* No installation required
* Just open the file and play
